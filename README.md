Private Chest est un gestionnaire de mots de passe local-first, open source, 
conçu pour stocker des identifiants, notes, cartes et documents dans un coffre chiffré sur l’ordinateur de l’utilisateur.
(Private Chest protège les données au repos, mais ne peut pas protéger un ordinateur déjà compromis par un malware, keylogger ou accès administrateur hostile.)
